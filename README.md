## 動機
  在高中教材的物理實驗中，談到測量荷質比需要一個均勻磁場，最好能消除掉地磁的效應。其中有個裝置吸引了我們的注意，就是赫姆霍茲線圈，一開始會有興趣純粹是因為名稱很像鋼之煉金術師裡的霍恩海姆，後來進一步的了解之後，想知道能夠做到讓磁場多均勻，如果使用交流電，又會有什麼現象。
